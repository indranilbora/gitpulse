# Support

## Getting Help

- Usage questions and bug reports: GitHub Issues
- Security issues: follow `SECURITY.md` private reporting process

## What to Include in Issues

- AgentPulse version (`agentpulse --help` output header or release tag)
- Platform details (`uname -a`, terminal, shell)
- Reproduction steps and expected vs actual behavior
- Relevant command output (`--once`, `--summary`, `--dashboard-json`) with secrets removed

## Scope

AgentPulse is macOS-first. Linux and Windows are best-effort and may have feature gaps.

## Response Model

This project is maintained with a solo-maintainer workflow. Critical regressions and security issues are prioritized first.
